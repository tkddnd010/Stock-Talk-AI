# 📈 Stock-Talk-AI
> **실시간 금융 데이터 기반 자율형 AI 에이전트 토론 플랫폼**

## 1. 개요
현재 개인 투자자의 시장 분석은 파편화된 정보를 수동으로 수집하는 데 머물러 있습니다. 본 프로젝트는 이러한 정보 분석 과정을 자동화하고, 가치 투자자, 기술적 분석가 성향을 가진 AI 에이전트들이 자율적으로 토론하여 인사이트를 제공하는 백엔드 시스템입니다.

## 2. 주요 기능
* **실시간 데이터 파이프라인:** 10분 주기 주가 및 뉴스 데이터 자동 수집
* **이벤트 기반 트리거:** 전일 대비 변동률  4%  이상 발생 시 AI 토론 자율 생성
* **비동기 작업 큐:** Redis 및 BullMQ를 활용한 LLM API 호출 최적화 및 서버 부하 분산
* **멀티 에이전트 페르소나:** 가치 투자자, 기술적 분석가 투자 관점 구현

## 3. 기술 스택 (선택 근거)
* **Framework:** NestJS (구조적 설계 및 유지보수성 고려)
* **Database:** PostgreSQL (정밀한 소수점 처리와 AI 분석을 위한 복잡한 쿼리 성능을 고려)
* **Infrastructure:** Redis / BullMQ (AI 연산의 비동기 처리 및 시스템 안정성 확보)
* **AI:** OpenAI GPT-4o mini (응답 속도 및 비용 효율성 최적화)

## 4. 데이터 목표
* **데이터셋 규모:** 1000건 이상 목표
* **분석 대상:** 해외 주식
